ZASADY dla wszystkich klas:

#1 nazwa KONTA ma składać się z Nazwiska i Imienia zgodnie ze schematem:
	* NazwiskoImie
	* Nazwisko.Imie
	* Nazwisko-Imie
#2 Repozytoria (REPO) mają być:
	* Publiczne lub
	* Prywatne z dodanym prowadzącym jako uprawniany do wglądu do REPO
#3 każde ZADANIE / Projekt wymaga założenia REPO zgodnie ze schematem
#4 schemat nazywania REPO to: PRZEDMIOT-ID.ZADANIA, np.:
	* AD-app-ide
	* AI-ajax
	* AI-min-proj
	* AM-sqlite-1
	* AM-sqlite-2
#5 każde REPO musi zawierać plik README z informacjami:
	* KLASA Nazwisko Imie
	* krótki opis zadania
	* JEŻELI zadanie tego wymaga - dodatkowe istotne informacje
